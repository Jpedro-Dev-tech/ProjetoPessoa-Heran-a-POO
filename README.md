# Projeto Pessoa - Herança em Java

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar os conceitos de Programação Orientada a Objetos (POO) em Java, com foco no uso de Herança.

A aplicação possui uma classe base chamada `Pessoa`, da qual derivam outras classes especializadas, demonstrando o relacionamento "é um" e a reutilização de código.

## 🎯 Objetivos

* Compreender o conceito de Herança em Java.
* Aplicar os princípios da Programação Orientada a Objetos.
* Reutilizar atributos e métodos por meio de classes derivadas.
* Organizar o código de forma mais estruturada e escalável.

## 🏗️ Estrutura do Projeto

### Classe Base

* **Pessoa**

  * Nome
  * Idade
  * DataNascimento
  * Sexo

### Classes Derivadas

  * Matrícula

* **Professor**
